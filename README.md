# UnPotty Python Games - Deluxe Prototype

This repository now contains a **new feature-heavy prototype** called **UnPotty Deluxe Super Mega Collection**.

## Included files

- `unpotty_deluxe.py` - Main pygame game loop with:
  - character selection (Toddler/Kid)
  - clothing removal selection
  - level selection
  - long scripted "mega un-potty" action sequence
  - target item roulette from a giant item list
- `game_data.py` - Story/event text and giant target list.
- `asset_factory.py` - Procedural sprite and simple tone generator.

## Run

```bash
python3 -m pip install pygame
python3 unpotty_deluxe.py
```

## Controls

- `1` / `2` select menu options
- `SPACE` advances the gameplay sequence
- `R` restart after completion
- `ESC` quit

This is intentionally comedic and exaggerated per the prompt request.
